# Portafolio Power BI

Este repositorio contiene un proyecto de Power BI con un dashboard interactivo para análisis de datos.

## 📊 Descripción

Dashboard de Power BI diseñado para visualizar y analizar datos de manera interactiva. Este proyecto demuestra capacidades de análisis de datos, visualización y storytelling con datos.

## 🚀 Características

- Visualizaciones interactivas
- Filtros dinámicos
- Análisis de datos en tiempo real
- Diseño responsivo
- Storytelling con datos

## 📋 Requisitos

- **Power BI Desktop** (versión gratuita o Pro)
- Windows 10 o superior
- Conexión a internet (para actualizaciones de datos si aplica)

## 💻 Instalación

1. Clona este repositorio:

   ```bash
   git clone https://github.com/kathepu/portafolio_power_bi.git
   ```

2. Navega al directorio del proyecto:

   ```bash
   cd portafolio_power_bi
   ```

3. Abre el archivo `presentación_final.pbix` con Power BI Desktop

## 📖 Uso

1. Descarga e instala [Power BI Desktop](https://www.microsoft.com/es-es/download/details.aspx?id=58494)
2. Abre el archivo `presentación_final.pbix`
3. Interactúa con los filtros y visualizaciones
4. Explora las diferentes páginas del dashboard

## 🛠️ Tecnologías

- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query (M)
